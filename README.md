### What's WSVuls?

WSVuls can find the busy requests from URL endpoints and mapping all requests from url.

with WSVuls we can extract:

- First Byte
- Start Render
- FCP
- Speed Index
- LCP 
- CLS
- TBT
- DC Time
- DC Requests
- DC Bytes
- Time
- Requests
- Total Bytes

in mapping features we can extract:

- Resource
- Request Start
- Content Type
- DNS Lookup
- SSL Negotiation
- Error/Status Code
